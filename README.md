# video-player-vue

> A video player plugin with vue2.

## public
``` bash
npm run build

npm public --access public

```
## Build Setup

``` bash
npm i @miyazhong/video-player-vue

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
